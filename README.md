# Projet Fauteuil Roulant Intelligent

Ce projet simule un fauteuil roulant intelligent capable de naviguer dans un environnement avec des obstacles et des humains, en utilisant des techniques de renforcement par l'apprentissage (Reinforcement Learning).

## Description

Ce projet vise à développer un environnement de simulation pour un fauteuil roulant intelligent. Le fauteuil doit naviguer vers un point d'arrivée (goal) en évitant les obstacles et les humains, qui peuvent être statiques ou dynamiques.

---

## Fonctionnalités

- Simulation d'un fauteuil roulant dans un environnement 2D.
- Génération aléatoire d'obstacles et d'humains (statiques et dynamiques).
- Contrôle du fauteuil via le clavier ou un joystick.
- Détection des collisions avec les obstacles et les humains.
- Visualisation en temps réel de l'environnement avec Pygame.
